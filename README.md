# 📦 CRUD Inventario — Node.js + MySQL

Sistema de gestión de inventario para tienda de productos electrónicos.

## Tecnologías
- Node.js + Express
- MySQL2
- Multer (subida de imágenes)
- HTML + CSS + JavaScript (vanilla)

## Instalación

1. Clona el repositorio
```bash
   git clone https://github.com/tu-usuario/crud-inventario-nodejs.git
   cd crud-inventario-nodejs
```

2. Instala dependencias
```bash
   npm install
```

3. Crea la base de datos ejecutando `database.sql` en phpMyAdmin

4. Inicia el servidor
```bash
   npm start
```

5. Abre http://localhost:3000

## Operaciones CRUD
- ✅ CREATE — Registrar nuevo producto con imagen
- ✅ READ — Listar y buscar productos
- ✅ UPDATE — Editar información y reemplazar imagen
- ✅ DELETE — Eliminar producto del inventario
